# EduPath LMS Dashboard UI Plan

This dashboard covers only internal authenticated LMS pages. It uses React, Tailwind CSS, React Router, Framer Motion, Lucide icons, and Recharts.

## Global Shell
- Desktop sidebar with internal scroll.
- Mobile drawer sidebar with overlay.
- Top navbar with search, notifications, theme toggle, avatar, and profile dropdown.
- Shared cards, headings, progress bars, empty/loading/error state patterns.

## Pages
1. Student Dashboard: welcome summary, profile card, stats, continue learning, charts.
2. My Courses: search, filters, progress course cards, empty state.
3. Course Learning: modules, lecture player, progress panel, actions.
4. Learning Paths: roadmap cards, progress tracker, skill readiness.
5. Quizzes: timer, questions, options, explanations, score result.
6. Assignments: upload box, list, deadline, grade, feedback.
7. Certificates: stats-style cards, preview/download/verify actions.
8. Achievements: badges, milestones, rewards.
9. Wishlist: saved learning cards and readiness actions.
10. Notes: course-based note cards, editor, save/delete actions.
11. Downloads: resource list with file details.
12. Calendar: schedule cards for classes, quizzes, deadlines.
13. Notifications: filter-ready feed with read/update items.
14. Messages: inbox-style feed.
15. Orders: responsive invoice table.
16. Profile: avatar, personal info, skills, bio.
17. Settings: account, notifications, security, theme preferences.
18. AI Tutor: chat UI, suggestions, course context, save answer.
19. AI Notes Summarizer: course/lecture selector, summary, key points.
20. AI Course Recommendation: interests, skills, career goal, course cards.
21. ML Analytics: engagement, completion probability, skill charts.
22. Community: discussion feed, replies, upvotes.
